# NVIM config

## Description

## Pre-requisites

### Linux

```bash
    sudo apt-get install -y neovim git luarocks
```

## Installation
Neovim and vim configuration files will be installed if Neovim or Vim is
installed on the system. The configuration files will be installed in the user's
home directory.

### Linux

```bash
    ./install.sh
```
### Windows

```bash
    ./install.bat
```
